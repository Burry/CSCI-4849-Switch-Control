About this app

This simple calculator demonstrates built-in switch control through use of the directional arrow and enter/return keys.

How to Use

Interact with the calculator by pointing and clicking on buttons with a mouse or trackpad or by focusing on buttons with the up/down/left/right directional arrow keys and selecting with the enter/return key.

User Testing

User testing was performed to determine the performance differences between the standard calculator (A) using all four directional arrow keys and a modified calculator (B) using only the left/right arrow keys for button focusing. 3 trials were performed on both A and B to input the following equations as fast as possible while still maintaining accuracy:
